<h1 align="center">Architecture</h1>

### Standard directory structure

```
├── public/
└── src /
	├── assets/                     // static resource & stypesheet CSS directory
    ├── types/                     	// Contain all types TypeScript
    ├── services/                   // pattern api call
    ├── components/                 // components(global & local) directory
    ├── containers/                 // Page components directory
    ├── themes/                     // contain all config of 3rd libs
    ├── helpers/                    // all functions type helper
    ├── templates/                  // Includes layout page
    ├── utils/                      // Tool function directory
    ├── App.tsx						// routing configuration Directory
    ├── index.tsx
    ├── rootReducer.ts
    ├── rootSaga.ts
    ├── store.ts					// State management directory
├── .env            
└── package.json
```
